ActionGraph API Reference

这里发布的是每次 GitHub Pages 部署时生成的 Javadoc。模块文档来自当前 main 分支，适合开发接入、代码评审和 1.0 兼容性冻结前的 API 对照。

机器可读 API surface 仍以 public-api.snapshot 为兼容性守门依据；Javadoc 用于阅读合同、线程安全、空安全和生命周期语义。
runtime

actiongraph-core

Action、Planner、Executor、Policy、Trace、Goal、Runtime API、Memory、Observability 与 API stability annotations。

 java8-client

actiongraph-control-plane-api

Java 8 控制面契约、组件目录、HTTP clients、properties 聚合配置与 shared-secret token verifier。

 review

actiongraph-human-review

审批任务、审批链、回调处理、Repository-backed policy 与任务查询/决策服务。

 governance

actiongraph-governance

脱敏、额度、权限、租户、审批属性和风险审批链路由策略。

 llm

actiongraph-llm-deepseek

实验态 LLM provider、prompt rendering、structured output parser 与 DeepSeek-compatible client。

 jdbc

actiongraph-persistence-jdbc

Trace、SuspendedRun、Memory、HumanReview 和 Console read model 的 JDBC 仓储。

 ops

actiongraph-console

只读运行查询、Trace 时间线响应、审计链验证结果和 CSV/JSONL 导出服务。

 spring

actiongraph-spring-boot-starter

Spring Boot auto-configuration、endpoint controllers、配置属性、JDBC/Memory/Review/Governance wiring。
